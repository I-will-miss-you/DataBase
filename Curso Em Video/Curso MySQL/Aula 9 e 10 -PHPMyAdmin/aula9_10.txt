-- No terminal

* Para ver todas as Bases de Dados
- show databases; 

* Verificar a Base de Dados aberta
- status;

* Abrir uma dada Base de Dados
- use cadastro;

* Ver as tabelas
- show tables;

* Ver a estrutura de uma dada tabela
- describe cursos;

* Ver o comando usado para criar uma dada tabela
- show create table amigos;

* Ver o comando usado para criar o Banco de Dados
- show create database exemplo;
